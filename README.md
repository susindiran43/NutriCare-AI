# NutriCare AI

AI-powered healthcare nutrition recommendation system built using RAG architecture.

## Features

- Disease prediction from symptoms
- Food recommendations
- Food avoidance suggestions
- RAG-powered knowledge retrieval
- Gemini AI response generation
- Streamlit dashboard

## Tech Stack

- Python
- Streamlit
- FAISS
- Sentence Transformers
- Gemini API
- Scikit-Learn

## Architecture

Symptoms → Disease Prediction → FAISS Retrieval → Gemini AI → Recommendations

## Installation

pip install -r requirements.txt

streamlit run app.py
